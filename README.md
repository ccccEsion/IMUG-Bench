# IMUG-Bench

**Coming soon.**

IMUG-Bench is a benchmark for evaluating multi-turn image generation and editing models. It focuses on instruction following, visual consistency, dynamic visual reasoning, and multi-turn dependency across diverse image-centric tasks.

We are preparing the public release of the benchmark data, evaluation protocol, and scoring tools.

## Status

This repository is currently a placeholder for the upcoming release. Data, scripts, documentation, and licenses are being organized for public use.
